# Release 1.1.3 - Camada oficial de ambiente

Data: 2026-06-09

## Tipo

Funcionalidade estrutural.

## Alterações

- Criado `clean/env.js`.
- Criado `index-teste.html`.
- Atualizado `clean/api.js` para reconhecer ambiente.
- RPCs de gravação passam a ser bloqueadas no modo `staging`.
- A versão visual passa a diferenciar produção e teste.

## Produção

```text
v1.1.3 - camada-env
```

## Teste

```text
v1.2.0-beta.1 - homologacao-free
```

## Regra

A URL de teste é:

```text
index-teste.html
```

Ela redireciona para:

```text
index-clean.html?env=staging
```

O modo staging exibe banner vermelho e bloqueia RPCs de gravação para proteger o banco de produção.

## Banco de dados

Nenhuma alteração de tabela, RPC ou policy foi executada nesta versão.
